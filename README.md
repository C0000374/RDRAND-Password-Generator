# RNDRAND Password Generator
Этот генератор паролей использует инструкцию RDRAND (появилась в Intel Ivy Bridge) для генерации пароля. Программа не будет работать на процессорах, которые не поддерживают эту инструкцию.
## Алгоритм работы генератора:
1. Генерируется 4 32-битных числа
2. Осуществляется проход по всем 16 полученным байтам, значение байта заменяется на ASCII-значение ````Table[Byte % 62]```` (Table - массив символов ASCII 0-9 A-Z a-z)
3. Полученные символы выводятся в STD_OUTPUT_HANDLE
